# Телемост на GitHub Issues

## Как настроить:

1. Создай Personal Access Token в GitHub:
   - Settings → Developer settings → Personal access tokens → Tokens (classic)
   - Выбери права: `repo` и `issues`
   - Скопируй токен

2. Открой файл `index.html` и вставь свои данные:
   - `GITHUB_REPO = 'popov2012/YourMom228_joke'`
   - `GITHUB_TOKEN = 'твой_токен'`

3. Загрузи папку на GitHub и включи Pages

## Как пользоваться:

1. Открой сайт → "Создать комнату"
2. Дай ссылку друзьям
3. Админ подтверждает вход ✅ или ❌
